* Pi Wake-On-LAN

Use Raspberry Pi to wake up WiFi-connected laptop.

** API

   #+BEGIN_SRC http :pretty
     GET http://127.0.0.1:5000/api/v1/machines
   #+END_SRC

   #+RESULTS:
   #+begin_example
   {
     "192.168.15.15": {
       "name": "fake",
       "ip": "192.168.15.15",
       "mac": "6c:40:08:8b:4b:02",
       "up": false
     },
     "192.168.15.31": {
       "name": "tung-mbp",
       "ip": "192.168.15.31",
       "mac": "6c:40:08:8b:4b:02",
       "up": true
     }
   }
   #+end_example

   #+BEGIN_SRC http :pretty

   #+END_SRC

   #+RESULTS:
